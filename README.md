# 운명의 바위 — 사주 파괴 클리커

모바일 방치형 클리커 게임 프로토타입 (HTML / CSS / JavaScript)

## 실행 방법

`index.html`을 브라우저에서 열거나, 로컬 서버로 실행하세요.

```bash
npx serve .
```

## 기능

- **운명의 바위**: SVG 다각형 + 네온 시안 테두리, 클릭 시 10% 확대·축소 + 빨간 테두리 플래시
- **의지 포인트**: 클릭 +1, 상단 HUD 실시간 표시, 터치 위치 `+1` 플로팅 텍스트
- **스테이지 클리어**: 목표(100 → 200 → 400) 달성 시 바위 파괴 이펙트 + 타이핑 팝업
- **업그레이드 스토어**: 실행력 강화(+1/초, 50P), AI 활용 능력(+5/초, 200P)
